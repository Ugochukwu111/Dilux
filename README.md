# DILUX Properties

## Overview

DILUX Properties is a premium real estate investment platform that connects high-net-worth investors with vetted land investment opportunities in Nigeria's most strategic locations (Epe, Ibeju-Lekki, and surrounding prime areas). The platform offers a transparent, frictionless pathway to acquire premium land assets with guaranteed 24% annual returns, complete documentation, and professional asset management.

## Who It's For

- **Individual Investors**: Looking for alternative investment vehicles with high, predictable returns
- **Family Offices**: Seeking to diversify portfolios into real estate with professional oversight
- **Corporate Entities**: Wanting to allocate capital into secured, documented land assets
- **International Investors**: Interested in Nigerian real estate opportunities with full compliance and transparency

## What We Solve

Traditional real estate investment in Nigeria is fragmented, opaque, and risky. DILUX addresses this by:

- **Eliminating Guesswork**: Free site inspections before any commitment. Investors see exactly what they're buying.
- **Building Trust**: Complete documentation with minimum two valid legal instruments per property. Full transparency in all processes.
- **Ensuring Security**: Professional due diligence, registered partnerships, and legal compliance backing every investment.
- **Simplifying Access**: Straightforward process from consultation to investment closure to automated dividend payouts.

## Quick Start

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm preview

# Run linter
npm run lint
```

The application will be available at `http://localhost:5173`.
